Document Portal

An AI-powered document processing platform built with Python, designed to integrate Large Language Models (LLMs), embedding models, 
and vector databases through a modular and scalable architecture.

Features
Multiple LLM integrations (OpenAI, Gemini, Claude, Groq, Hugging Face, Ollama)
Embedding model support
Vector database integration
Centralized configuration management
Custom logging and exception handling
Dockerized deployment
Automated testing with PyTest

Tech Stack
Python
FastAPI
Large Language Models (LLMs)
Retrieval-Augmented Generation (RAG)
Vector Databases
Embedding Models
Docker
PyTest


Project Structure
api/            # API layer
config/         # Configuration
model/          # Model integrations
prompt/         # Prompt management
src/            # Core application logic
utils/          # Helper utilities
tests/          # Test cases


Getting Started
git clone https://github.com/nithinreddynarala/document_portal.git
cd document_portal
pip install -r requirements.txt
